# DataPulse Analytics

Enterprise Analytics & ML Platform built with Python, Gradio, scikit-learn, and Plotly.

## Features

- Player Performance Prediction
- User Risk Scoring
- Market Expected Value Analysis
- Interactive Data Exploration

## Running Locally

```bash
pip install -r requirements.txt
make build          # generate data + train models
python huggingface/app.py
```
